# Intro

<h4>홈 베이킹에 관심이 많은 당신? Bakers101로 오라! </h4>

Bakers101은 직접 빵을 굽는 '빵굽족'들을 위한 사용자 참여형 **SNS**로서,<br/>
당신의 관심 있는 빵을 쉽게 찾을 수 있게 도와드리고 있어요!
